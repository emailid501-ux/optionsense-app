üéØ What is OptionSense?

OptionSense is a real-time stock analysis tool that helps you:

  • See Top 100 NSE stocks with Buy/Sell recommendations
  • Get technical analysis (RSI, MACD, Fibonacci levels)
  • Search any NSE listed stock for analysis
  • Get Option buying strategies for weekly trades

üíª System Requirements

Component Minimum
OS Windows 10/11
RAM 4 GB
Python 3.9 or higher
Internet Required (for live data)

üì• Installation (One-Time Setup)

Step 1: Install Python

  1. Go to python.org/downloads
  2. Download Python 3.11 (or latest)
  3. IMPORTANT: Check ‚úÖ "Add Python to PATH" during installation
  4. Click "Install Now"

Step 2: Download OptionSense

  1. Extract optionsense.zip to a folder (e.g., C:\OptionSense)
  2. You should see two folders: backend and frontend

Step 3: Install Dependencies

  1. Open Command Prompt (Press Win + R, type cmd, press Enter)
  2. Navigate to backend folder: 
    cd C:\OptionSense\backend
  3. Install required libraries: 
    pip install -r requirements.txt
  4. Wait for installation to complete (2-5 minutes)

üöÄ How to Start the App

Step 1: Start the Server

  1. Open Command Prompt
  2. Go to backend folder: 
    cd C:\OptionSense\backend
  3. Start the server: 
    uvicorn main:app --host 0.0.0.0 --port 8000
  4. Wait until you see: Uvicorn running on http://0.0.0.0:8000

Step 2: Open the App

  1. Open Google Chrome browser
  2. Type this URL: http://localhost:8000/frontend/index.html
  3. Press Enter

üéâ Your app is now running!

üì± How to Use

Dashboard Tab

  • Shows NIFTY/BANKNIFTY sentiment (Bullish/Bearish/Neutral)
  • PCR (Put-Call Ratio) indicates market direction
  • VWAP Signal shows intraday trend

Stocks Tab

  • Buy (Green) Recommended to buy
  • Sell (Red) Recommended to sell/avoid
  • Watch (Yellow) No clear signal, monitor

Search Any Stock

  1. Type stock symbol in search box (e.g., CDSL, IRCTC, TITAN)
  2. Press Enter or click Search
  3. Analysis will appear within 10-30 seconds

Option Strategy

  • Click "Get 1-Week Option Strategy" on any stock card
  • Shows recommended Call/Put option to buy
  • Displays Strike Price, Trend, and Risk Level

üìö Trading Strategies Guide

1. PCR (Put-Call Ratio)

Formula: Total Put OI √∑ Total Call OI

PCR Value Signal Action
> 1.0 (e.g., 1.25) Bullish Buy on Dips
< 0.7 Bearish Sell on Rise
0.7 - 1.0 Neutral Wait for clarity

Tip: If PCR was 1.5 and dropped to 1.2, reversal may be coming!

2. OI Shift Analysis

Check every 15-30 minutes for shifts in Open Interest.

  • Support Shift Up (Bullish): Max Put OI moved from 24,000 ‚Üí 24,100. Market is bullish!
  • Resistance Shift Down (Bearish): Max Call OI moved from 24,500 ‚Üí 24,400. Bears are in control!
  • Unwinding: If OI is decreasing at a strike, short covering may push price up.

3. VIX (Volatility Index)

VIX Level Status Strategy
< 15 Low Option Buying is good
15 - 18 Moderate Be cautious
> 18 High Prefer Option Selling (premium decays fast)

4. Volume Analysis

  • True Move: Price UP + Volume UP = Real buying. Stay in trade!
  • Trap Move: Price making new high but Volume dropping = Fake breakout. Prepare for reversal!

5. OI Ladder

Identifies support and resistance levels based on OI concentration.

RESISTANCE (Call OI):
  24,500 ‚ñà‚ñà‚ñà‚ñà‚ñà‚ñà‚ñà‚ñà 50L OI (Strongest)
  24,600 ‚ñà‚ñà‚ñà‚ñà‚ñà‚ñà   40L OI
  24,700 ‚ñà‚ñà‚ñà‚ñà     30L OI

SUPPORT (Put OI):
  24,000 ‚ñà‚ñà‚ñà‚ñà‚ñà‚ñà‚ñà‚ñà 45L OI (Strongest)
  23,900 ‚ñà‚ñà‚ñà‚ñà‚ñà    35L OI

Use: Market will struggle to break these levels. Use for stop-loss and target planning.

6. Theta Decay (Straddle Tracking)

Add ATM Call price + ATM Put price = Straddle Price

  • If Straddle price is falling fast ‚Üí Market is sideways. Don't buy options!
  • If Straddle price is stable/rising ‚Üí Market is trending. OK to buy options.

7. Market Breadth (Advance/Decline)

Count how many NIFTY 50 stocks are up vs down.

Advancing Declining Signal
35+ < 15 Bullish - Look for Calls
< 15 35+ Bearish - Look for Puts
~25 ~25 Neutral - Avoid trading

8. Pro Trader Workflow (Morning Checklist)

Follow this routine every trading day:

  1. 9:15-9:30: Check Market Breadth (Advance/Decline)
  2. 9:30: Check PCR (> 1 = Bullish, < 1=Bearish)
  3. Before Entry: Confirm price is above/below VWAP
  4. Check OI: See if writers are exiting your direction
  5. During Trade: Monitor Volume and VIX

Using all 8 indicators together increases accuracy and reduces fake trades!
